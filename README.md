## Features <a name = "table"></a>

- Create account
- Sign in
- Publish post
- Edit posts
- See collection of posts
- See all users and accounts
- Likes for posts

## Installation <a name = "install"></a>

```sh
git clone https://github.com/Taramin/PublicDiary
cd {project-name}
./mvnw spring-boot:run
java -jar target/{project-name}.jar
```
